# GATE_fan_beam_simulation
Codes used in the paper "Simulating Equidistant Fan-Beam Binary Tomography for 2D Voxel Slices Using GATE 9.x" by Péter Balázs, Tibor Novák, Károly Osvay

The GATE scripts are written in GATE v9.2
